# 🏠 Airbnb Market Analytics Dashboard

**Interactive Tableau project visualizing Airbnb listing data to explore pricing and neighborhood trends.**

---

## 📖 Overview
This Tableau dashboard provides a clear, visual overview of Airbnb market dynamics across neighborhoods.  
It highlights how **location**, **property type**, and **average price** interact, helping identify patterns and potential investment opportunities.

The project was built to demonstrate data visualization, storytelling, and dashboard design skills using real-world Airbnb data.

---

## 📊 Features
- 📍 **Price by Neighborhood:** Interactive map and chart showing price variations across different areas.  
- 📈 **Comparative Analysis:** Visual comparison of listing prices and market averages.  
- 🎛️ **Dynamic Filters:** Drill down by neighborhood, room type, or price range.  
- 💡 **Actionable Insights:** Identify areas with the best value or highest returns.  

---

## 🧰 Tools Used
- **Tableau Public / Tableau Desktop**
- **Airbnb Open Data** (CSV format)
- **Excel** *(data cleaning and preprocessing)*


---

## 🚀 How to Open
1. Download the `.twbx` file from this repository.  
2. Open it in **Tableau Desktop** or **Tableau Public** (free to use).  
3. Explore the interactive dashboard and insights.
